# 百度搜不到爱奇艺了？爱奇艺副总裁回应：出现了bug，目前已恢复

![6e708cb2f233d3ff02ae6559e451a58f.jpg](./百度搜不到爱奇艺了爱奇艺副总裁回应出现了bug目前已恢复/6e708cb2f233d3ff02ae6559e451a58f.jpg)

Tech星球5月23日消息，今日晚间，百度搜不到爱奇艺了的消息引发热议。对此，爱奇艺副总裁邱建雄在朋友圈回应，没有任何问题，百度出现了bug正在解决中。

此前有网友反馈，通过百度搜索“爱奇艺”，结果显示“抱歉没有找到与爱奇艺相关的网页”。

![ffd895cbb0d0b309de736da1e4a47ab0.jpg](./百度搜不到爱奇艺了爱奇艺副总裁回应出现了bug目前已恢复/ffd895cbb0d0b309de736da1e4a47ab0.jpg)

小编实测发现，百度目前已正常恢复了“爱奇艺”搜索相关内容。
